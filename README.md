# 僕ラブコンテンツフィルター

[僕ラブのサークルリスト](http://www.puniket.com/lovelive/bokulove38_list02.htm)で特定のシリーズ(サンシャイン、虹ヶ咲など)に配置されたサークルだけをフィルタリングして表示します。  

## 使い方

ページ上部の** Code **のところから`Download ZIP`でZIPをダウンロードして解凍してください。
